# 欢迎使用 Muse！

这是一个示例文档，展示 Muse 插件的各种功能。

## Markdown 语法支持

### 标题

使用 `#` 创建标题，从 `#` 到 `######` 共六级标题。

### 文字样式

- **粗体文本** 使用 `**粗体**`
- *斜体文本* 使用 `*斜体*`
- ~~删除线~~ 使用 `~~删除线~~`
- `代码` 使用反引号

### 列表

无序列表：
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3

有序列表：
1. 第一步
2. 第二步
3. 第三步

## 使用技巧

1. **保存文件**: 输入文件名后点击保存按钮
2. **预览模式**: 点击预览按钮启用分屏显示
3. **Git 提交**: 初始化后可以提交和查看历史
4. **快速访问**: 点击浏览器工具栏图标快速打开
