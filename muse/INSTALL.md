# 快速安装指南

## 方法一：直接加载（推荐）

图标已经生成完毕，可以直接安装：

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 打开右上角的 **"开发者模式"** 开关
4. 点击左上角的 **"加载已解压的扩展程序"** 按钮
5. 选择 `muse` 文件夹
6. 完成！插件图标会出现在浏览器工具栏

## 方法二：重新生成图标

如果需要自定义图标，可以：

### 使用 Python 脚本（推荐）

```bash
cd muse
python3 generate-icons.py
```

### 使用浏览器工具

1. 在浏览器中打开 `create-icons.html`
2. 点击"下载图标"按钮
3. 将下载的文件移动到 `icons/` 目录

## 使用插件

1. 点击浏览器工具栏中的插件图标（蓝色 M 图标）
2. 弹出编辑器窗口
3. 开始编写 Markdown！

### 首次使用

1. 输入文件名，例如：`note.md`
2. 在编辑器中输入 Markdown 内容
3. 点击 💾 保存
4. 点击 🔧 Init 初始化 Git
5. 输入提交信息，点击 ✅ Commit

## 验证安装

安装成功后，您应该看到：

- ✅ 浏览器工具栏出现蓝色 M 图标
- ✅ 点击图标打开编辑器窗口
- ✅ 窗口大小为 800x600 像素
- ✅ 所有按钮和功能正常显示

## 故障排除

### 插件无法加载
- 确认已开启"开发者模式"
- 检查 manifest.json 是否存在
- 查看是否有错误提示

### 预览不工作
- 检查 libs/marked.min.js 是否存在
- 打开 Chrome 开发者工具（F12）查看错误

### 图标不显示
- 确认 icons 目录下有 icon16.png、icon48.png、icon128.png
- 运行 generate-icons.py 重新生成图标

## 卸载

1. 访问 `chrome://extensions/`
2. 找到 "Muse"
3. 点击"移除"按钮

**注意**: 卸载将删除所有已保存的文件和 Git 历史！
